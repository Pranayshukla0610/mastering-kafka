# mastering-kafka
A real-time data streaming project demonstrating Apache Kafka fundamentals, distributed messaging, producer-consumer architecture, and event-driven Data Engineering workflows.

---

## Tech Stack

- Apache Kafka
- Zookeeper
- Python
- Docker
- Docker Compose
- PostgreSQL
- Pandas

---

## Features

- Real-time data streaming
- Kafka producer-consumer workflows
- Event-driven architecture
- Topic management
- Stream processing
- Fault-tolerant messaging
- Dockerized Kafka setup
- Scalable distributed pipelines
